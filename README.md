# What is this

Style Wizard

# Installation

`npm i stylewizard --save`

Then

```import {stylewizard} from 'stylewizard;

   stylewizard({
       shadow_type: 'soft',
       padding:'false'
   });
```

## options

Style wizard supports 2 options, both of which are optional:

- _shadow_type_ - _hard | soft_ (Defaults to soft)

- _padding_ - _boolean_ (Defaults to false)
